## 2. CodePilot 차별화 포인트 (강점)

| 항목 | 내용 |
|------|------|
| **멀티 에이전트 오케스트레이션** | 병렬 Investigation + Sequential Execution, 시장에서 거의 없음 |
| **팀 RAG (사내 문서)** | 서버 기반 조직 단위 문서 검색, B2B 차별점 |
| **관리자 콘솔 + 설정 동기화** | AI 모델/프롬프트/보안규칙을 조직 단위로 강제 설정 가능 |
| **다국어 UI (7개국어)** | 한국어 포함 글로벌 B2B 대응 |
| **Hot Load** | 키워드 트리거 자동화, 독자적 기능 |
| **턴 단위 Diff 체크포인트** | 특정 턴까지 전체 롤백 가능 |
| **의도 분석 + 계획 수립** | 실행 전 사용자에게 작업 계획 투명하게 노출 |
