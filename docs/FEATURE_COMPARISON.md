## 2. CodePilot 차별화 포인트 (강점)

| 항목 | 내용 |
|------|------|
| **멀티 에이전트 오케스트레이션** | 병렬 Investigation + Sequential Execution, 시장에서 거의 없음 |
| **팀 RAG (사내 문서)** | 서버 기반 조직 단위 문서 검색, B2B 차별점 |
| **관리자 콘솔 + 설정 동기화** | AI 모델/프롬프트/보안규칙을 조직 단위로 강제 설정 가능 |
| **다국어 UI (7개국어)** | 한국어 포함 글로벌 B2B 대응 |
| **Hot Load** | 키워드 트리거 자동화, 독자적 기능 |
| **턴 단위 Diff 체크포인트** | 특정 턴까지 전체 롤백 가능 |
| **의도 분석 + 계획 수립** | 실행 전 사용자에게 작업 계획 투명하게 노출 |

## 3. 실제 제품 납품을 위해 필요한 것

### 3-1. 핵심 기능 (없으면 경쟁 불가)

| 항목 | 우선순위 | 설명 |
|------|:--------:|------|
| **인라인 자동완성 (Tab)** | 🔴 최우선 | Cursor/Copilot의 가장 많이 쓰는 기능. 없으면 "AI 에디터"로 안 보임. `vscode.languages.registerInlineCompletionItemProvider` |
| **인라인 편집 (Cmd+K)** | 🔴 최우선 | 에디터 내 선택 → 즉시 수정. 채팅 전환 없는 빠른 편집 UX |
| **코드베이스 벡터 인덱싱** | 🔴 최우선 | 대규모 프로젝트에서 관련 파일 자동 찾기. 현재 RelevantFilesFinder는 텍스트 검색만 |
| **프롬프트 캐싱** | 🟠 높음 | Anthropic/OpenAI 프롬프트 캐싱으로 반복 요청 비용 60-80% 절감. B2B 수익성 필수 |

### 3-2. 안정성 / 신뢰성

| 항목 | 우선순위 | 설명 |
|------|:--------:|------|
| **Rate Limiting + 사용자 쿼터** | 🔴 최우선 | 팀 플랜에서 API 비용 통제. 현재 없음 |
| **LSP 완성** | 🟠 높음 | Go-to-definition, hover docs를 컨텍스트로 활용. 진단 정확도 향상 |
| **테스트 코드** | 🟠 높음 | 핵심 로직(ConversationManager, ToolExecutor) 단위 테스트 전무 |
| **오프라인 모드** | 🟡 중간 | 네트워크 단절 시 로컬 Ollama만으로 기본 동작 보장 |

### 3-3. UX / 온보딩

| 항목 | 우선순위 | 설명 |
|------|:--------:|------|
| **온보딩 위저드** | 🟠 높음 | 첫 설치 → API 키 설정 → 첫 채팅까지 가이드. 현재 없음 |
| **키보드 단축키 커스터마이즈** | 🟡 중간 | 수락/거절 단축키 등 설정 가능하게 |
| **채팅 메시지 편집/재전송** | 🟡 중간 | Cursor·Cline 모두 지원. 이전 질의 수정 재전송 |
| **마크다운 Export** | 🟢 낮음 | 대화 내보내기 |

### 3-4. B2B/엔터프라이즈

| 항목 | 우선순위 | 설명 |
|------|:--------:|------|
| **SSO (SAML/OIDC)** | 🟠 높음 | 기업 고객 필수 요건 |
| **감사 로그 (Audit Log)** | 🟠 높음 | 어떤 코드가 AI로 생성됐는지 기록 |
| **데이터 레지던시** | 🟡 중간 | EU/국내 데이터가 외부 LLM에 안 나가는 옵션 (on-premise Ollama로 일부 대응) |
| **팀 사용량 대시보드** | 🟡 중간 | 관리자가 팀원별 사용량/비용 확인 |
| **코드 보안 스캔 연동** | 🟢 낮음 | Snyk/SonarQube 연동으로 AI 생성 코드 취약점 검사 |

## 4. 요약

CodePilot은 **B2B SaaS AI 코드 어시스턴트**로 차별화된 포지션을 갖고 있습니다.
팀 RAG, 관리자 콘솔, 멀티 에이전트, 다국어 UI는 독자적 강점이지만,
**인라인 자동완성과 인라인 편집 미구현**이 가장 큰 시장 진입 장벽입니다.

납품 최소 요건: 인라인 완성 + 벡터 인덱싱 + 프롬프트 캐싱 + Rate Limiting
