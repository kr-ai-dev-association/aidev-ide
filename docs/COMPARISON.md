## CODEPILOT 핵심 차별점

### 1. 작업별 모델 분리 (Unique)
- **대화 압축용**: 빠르고 저렴한 모델로 토큰 절약
- **명령어 생성용**: 코드 특화 모델로 정확한 터미널 명령
- **의도 분석용**: 경량 모델로 빠른 사용자 의도 파악
- 각 작업에 최적화된 모델을 선택하여 **비용 절감 + 성능 향상**

### 2. 완전한 자동화 파이프라인
- 자동 에러 수정: 최대 10회 재시도
- 자동 테스트 재시도: 빌드/린트 실패 시 자동 수정
- 터미널 명령 자동 실행 (선택적 승인)

### 3. 기업 환경 최적화
- **온프레미스 배포**: 외부 API 없이 Ollama만으로 운영
- **Banya 자체 모델**: 국내 기업용 LLM 연동 (Solar 100B, Qwen-Coder 32B)
- **API 키 보안 저장**: VSCode Secret Storage 활용

### 4. 세션 연속성
- VSCode 재시작 후에도 대화 히스토리 복원
- 프로젝트 컨텍스트 캐싱으로 빠른 응답

### 5. 커스텀 개발 규칙
- 프로젝트별 개발 규칙 MD 파일 지원
- 코딩 컨벤션, 아키텍처 가이드 등을 AI에게 전달
- 팀 전체 일관된 코드 스타일 유지

### 6. 글로벌 지원
- 7개 언어 UI: 한국어, 영어, 중국어, 일본어, 독일어, 프랑스어, 스페인어
