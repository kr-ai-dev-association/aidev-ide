## 유지된 핵심 기능

- CODE 모드 에이전트 루프 (Investigation → Execution → Review → Done)
- 멀티 에이전트 오케스트레이션 (TaskSplitter → SubAgentLoop 병렬 실행)
- 도구 실행 (파일 읽기/쓰기/생성/삭제, 터미널 명령, 검색 등)
- 스트리밍 즉시 실행 (LLM 응답 실시간 표시 + 파일 생성/수정 즉시 적용)
- 보안 규칙 (프로젝트 외부 파일 차단, 위험 명령 패턴 차단)
- 자동 테스트/에러 수정 재시도
- 인라인 코드 완성
- 세션 관리 및 히스토리
- 모델 라우팅 (Ollama/OpenAI/Anthropic/Gemini)
- Hot Load 기능
- 다국어 지원
