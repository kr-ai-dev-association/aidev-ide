# Codepilot (Standalone)

AI-powered development assistant for VS Code with multi-LLM support.

## Features

- **Multi-LLM Support** — Ollama (local), OpenAI, Gemini, Anthropic
- **Agent Mode** — 5-stage FSM loop: Investigation > Plan > Execution > Review > Done
- **Multi-Agent Orchestration** — Auto-splits complex tasks into parallel sub-agents
- **14+ Built-in Tools** — File CRUD, ripgrep search, glob search, command execution, LSP, code definitions
- **Inline Diff Review** — Code changes shown as inline diffs in editor with per-turn undo
- **Auto Test & Build** — Automatic verification after code generation with retry
- **Inline Completion** — Ghost text suggestions powered by LLM
- **Cross-Platform** — Windows / macOS / Linux with OS-specific adapters

## Getting Started

1. Install the extension from VS Code Marketplace
2. Open Settings and configure your LLM provider (Ollama, OpenAI, Gemini, or Anthropic)
3. Start coding with AI assistance in the chat panel
