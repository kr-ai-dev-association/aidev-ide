# AgentGoCoder

VS Code용 AI 코딩 어시스턴트 — Ollama, OpenAI, Gemini, Anthropic 등 여러 LLM을 선택해 사용할 수 있습니다.

## Features

- **Multi-LLM** — Ollama(로컬), OpenAI, Gemini, Anthropic
- **CODE 워크플로** — Investigation → Execution → Review → Done 단계 기반 에이전트 루프
- **멀티 에이전트 오케스트레이션** — 복잡한 작업을 병렬 서브 에이전트로 분할 실행
- **내장 도구** — 파일 CRUD, ripgrep/glob 검색, 터미널 명령, LSP, 정의 탐색 등
- **인라인 Diff 검토** — 에디터에서 변경 diff 확인 및 턴 단위 실행 취소
- **자동 테스트·빌드** — 생성 후 검증 및 재시도
- **인라인 완성** — LLM 기반 고스트 텍스트 제안
- **크로스 플랫폼** — Windows / macOS / Linux

## Getting Started

1. VSIX로 설치하거나 마켓플레이스에서 **AgentGoCoder** 확장을 설치합니다.
2. 설정에서 사용할 LLM 공급자(Ollama, OpenAI, Gemini, Anthropic 등)를 구성합니다.
3. 채팅 패널에서 질문·코드 생성·리팩터링 등에 활용합니다.
