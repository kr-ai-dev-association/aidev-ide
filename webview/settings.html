LLM이 제안한 코드를 기반으로 파일을 자동으로 업데이트할지 여부를 설정합니다. 비활성화 시, 업데이트 전에 확인 메시지가 표시됩니다.

OUTPUT 로그 제어

VS Code의 OUTPUT 패널에 표시되는 로그를 활성화/비활성화할 수 있습니다. 비활성화하면 터미널 로그 확인이 더 편해집니다.

자동 오류 수정

명령어 자동 실행

LLM 응답에 포함된 bash, powershell, cmd 명령어를 자동으로 실행할지 여부를 설정합니다. 비활성화하면 수동으로 실행해야 합니다.

AI 모델 설정

CODEPILOT의 AI 기능을 사용하기 위한 모델 설정입니다. 사용할 AI 모델을 선택하고 해당 설정을 구성하세요.

Gemini 설정

Google의 Gemini API를 사용하여 AI 코드 생성 및 분석 기능을 활성화합니다.

등록 방법: Google AI Studio API 키 페이지에서 새 API 키를 생성하세요.

Ollama 설치

Ollama를 설치하여 로컬에서 LLM 모델을 실행할 수 있습니다.

설치 방법: 아래 링크에서 운영체제에 맞는 Ollama 설치 파일을 다운로드하여 설치하세요.

설치 후 터미널에서 ollama --version 명령어로 설치를 확인하세요.

Local Ollama 설정

로컬 머신에 설치된 LLM 모델을 사용하여 AI 코드 생성 및 분석 기능을 활성화합니다.

설정 방법: 로컬 머신에서 실행 중인 Ollama 서버의 HTTP API 주소를 입력하세요. (예: http://localhost:11434)

🌐 원격 Ollama 서버 설정

원격 서버에 설치된 LLM 모델을 사용하여 AI 코드 생성 및 분석 기능을 활성화합니다.

설정 방법: 원격 서버의 HTTP API 주소, 모델명, 엔드포인트를 입력하세요.

Ollama 모델 다운로드

로컬 Ollama에 사용할 수 있는 다양한 LLM 모델을 다운로드할 수 있습니다.

사용 방법: 원하는 모델의 다운로드 버튼을 클릭하면 자동으로 Ollama에 모델이 다운로드되고 드롭다운에 추가됩니다.
