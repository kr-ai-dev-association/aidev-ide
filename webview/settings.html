멀티 에이전트

복잡한 작업을 여러 에이전트가 병렬로 분담하여 처리합니다. 토큰 소모량이 단일 에이전트 대비 많아질 수 있습니다. 비활성화 시 단일 에이전트 방식으로 동작합니다.

파일 자동 업데이트

LLM이 제안한 코드를 기반으로 파일을 자동으로 업데이트할지 여부를 설정합니다. 비활성화 시, 업데이트 전에 확인 메시지가 표시됩니다.

자동 파일 삭제

LLM이 요청한 파일 삭제를 자동으로 수행할지 여부를 설정합니다. 비활성화 시, 파일 삭제 전에 확인 메시지가 표시됩니다.

도구 자동 실행

모든 도구(파일 읽기, 검색 등)를 자동으로 실행할지 여부를 설정합니다. 비활성화하면 모든 도구 실행 전에 확인 메시지가 표시됩니다.

MCP 도구 자동 실행

MCP 도구를 승인 없이 자동 실행할지 여부를 설정합니다. 비활성화하면 MCP 도구 실행 전에 확인 메시지가 표시됩니다.

명령어 자동 실행

LLM 응답에 포함된 bash, powershell, cmd 명령어를 자동으로 실행할지 여부를 설정합니다. 비활성화하면 명령어 실행전전에 확인 메시지가 표시됩니다.

자동 테스트 재시도

자동 테스트 검증(Smoke Test, Diagnostics, Lint Check)이 실패했을 때 자동으로 재시도하는 최대 횟수를 설정합니다. 관리자 설정(빌드/테스트)에 검증 명령어가 등록되어 있으면 해당 명령어가 우선 실행됩니다.
