시리얼 번호

시리얼 번호를 입력하여 제품을 활성화하세요.

파일 자동 업데이트

LLM이 제안한 코드를 기반으로 파일을 자동으로 업데이트할지 여부를 설정합니다. 비활성화 시, 업데이트 전에 확인 메시지가 표시됩니다.

자동 파일 삭제

LLM이 요청한 파일 삭제를 자동으로 수행할지 여부를 설정합니다. 비활성화 시, 파일 삭제 전에 확인 메시지가 표시됩니다.

도구 자동 실행

모든 도구(파일 읽기, 검색 등)를 자동으로 실행할지 여부를 설정합니다. 비활성화하면 모든 도구 실행 전에 확인 메시지가 표시됩니다.

명령어 자동 실행

LLM 응답에 포함된 bash, powershell, cmd 명령어를 자동으로 실행할지 여부를 설정합니다. 비활성화하면 명령어 실행전전에 확인 메시지가 표시됩니다.

자동 오류 수정

터미널에서 오류가 발생했을 때 자동으로 수정을 시도하는 최대 횟수를 설정합니다. (1-10회, 기본값: 5회)

회

설치 방법: 아래 링크에서 운영체제에 맞는 Ollama 설치 파일을 다운로드하여 설치하세요.

설치 후 터미널에서 ollama --version 명령어로 설치를 확인하세요.

Local Ollama 설정

로컬 머신에 설치된 LLM 모델을 사용하여 AI 코드 생성 및 분석 기능을 활성화합니다.

설정 방법: 로컬 머신에서 실행 중인 Ollama 서버의 HTTP API 주소를 입력하세요. (예: http://localhost:11434)

원격 Ollama 서버 설정

원격 서버에 설치된 LLM 모델을 사용하여 AI 코드 생성 및 분석 기능을 활성화합니다.

설정 방법: 원격 서버의 HTTP API 주소, 모델명, 엔드포인트를 입력하세요.

모델 라우팅 설정

사용자 쿼리의 의도를 분석하는 데 사용됩니다. 빠르고 저렴한 모델 권장.
